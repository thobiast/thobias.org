Há uns três anos, enxerguei a luz e conheci a doutrina dos amantes do
shell. Esta é a parte dentro da minha página dedicada a shell script.
Aqui vou colocar apenas códigos. Para textos e documentos procure no
diretório doc.

Existe uma excelente lista de discussão em português sobre este tema.
A lista não tem um tráfego muito alto e o nível é muito bom. Página da
lista http://br.groups.yahoo.com/group/shell-script/

Sobre sed, tem a lista de discussão sed-br (muito boa).

===========================================================

visitas.cgi é um Livro de Visitas em CGI escrito em bash que utilizo na minha página.

===========================================================

gravar é um script escrito em bash que utilizo para gravar meus CDs.
ele grava CD de áudio e dados.
grava dados a partir de uma imagem iso ou um diretório.
grava áudio a partir de mp3, wav ou ainda ripa um CD e depois grava.
apaga conteúdo de um CD regravável.

===========================================================

sig_dc é um script que fiz para gerar os 'números mágicos' que o DC
utiliza para escrever uma frase. Fazer na mão os cálculos da
muito trabalho. =8)


===========================================================

matematica é um pequeno script para treinar a matemática básica.
faz perguntas de + - / *, e depois te mostra uma pequena estatística
com o número de acertos e erros.

Acho que não serve pra nada, mas foi legal fazer =8)

===========================================================

einstein Hei, se você é mais velho
deve se lembra do programa einstein que rolou na internet há alguns
anos atrás. Aquele que você escrevia a resposta enquanto aparecia uma
frase na tela, e depois fazia a pergunta para a resposta ser a que
você digitou. Muito massa, me diverti muito com este programa.
Como a versão que eu tinha era para windows e eu perdi esta
partição da minha máquina de casa, escrevi uma versão em shell script
Ficou palha mas ficou massa! =8)

===========================================================

bashwser é um browser feito em bash e somente bash.
não serve pra nada, pois você não vai deixar de usar lynx ou algum
outro browser como mozilla, galeon...
mas serve de exemplo de como fazer um cliente servidor em shell script

no fonte coloquei um exemplo

===========================================================

conta_palavra.sed.html mail que enviei para a lista sed-br
demonstrando um exemplo de como usar o sed para contar.

===========================================================

txt2html.sed.html converte um arquivo texto em html.

===========================================================

identar.sed é um script sed para identar shell script. É palha!!

===========================================================

colorize_c.sed script para transformar código C em código colorido em HTML (2html).
PS: o VIM já faz isso (ls /usr/share/vim/vim*/syntax/2html.vim). Mas este script é em sed =8)
